# graffiti

> A single-stroke shorthand handwriting recognition system.

[![js-standard-style](https://cdn.rawgit.com/feross/standard/master/badge.svg)](https://github.com/feross/standard)

## Usage

Starting the server:

```sh
npm start
```

Testing the sources:

```sh
npm test
```
